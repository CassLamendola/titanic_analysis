The first step to clean up the data was to get rid of any data points with missing data. I did this by first dropping the columns for 'Ticket' and 'Cabin' because they contained the most missing data and I didn't think they were necessary for the analysis. I also dropped other data points (Name', 'PassengerId', 'SibSp' - siblings/spouse onboard, 'Parch' - parents/children onboard) that I did not plan to use in my analysis in order to make my data cleaner. Then, I dropped all of the rows (passengers) that had missing data. Several passengers were missing age information, and since my plan was to analyze how age, class, and gender affected survival rate, I thought it was best to remove those passengers altogether.

# calculates the survival rate (percentage of passengers who survived the sinking) for a given age range
def get_survival_by_age_range(lower_bound, upper_bound):
    age_group = titanic_df[titanic_df['Age']<upper_bound][(titanic_df['Age']>=lower_bound)]
    return (float(age_group['Survived'].sum())/len(age_group)) * 100

# calculates age range for a single passenger
def add_age_range(passenger):
    if passenger < 10:
        return '0 - 9'
    elif passenger < 20 and passenger >= 10:
        return '10 - 19'
    elif passenger < 30 and passenger >= 20:
        return '20 - 29'
    elif passenger < 40 and passenger >= 30:
        return '30 - 39'
    elif passenger < 50 and passenger >= 40:
        return '40 - 49'
    elif passenger < 60 and passenger >= 50:
        return '50 - 59'
    elif passenger < 70 and passenger >= 60:
        return '60 - 69'
    else:
        return '70+'

# adds new index and applies age range to all passengers
titanic_df['Age Range'] = titanic_df['Age'].apply(add_age_range)

# group data by age range and find mean survival
age_group = titanic_df.groupby('Age Range')['Survived'].mean() * 100

Conclusions:¶

Correlations exist between:

  • age and survival
  • class and survival

There also seems to be a connection between gender and survival rate. Women had higher survival rates than men. However, these correlations do not imply causation. Because I removed several passengers due to missing data, the findings are less accurate. Additionally, all findings are pending further statistical testing.
